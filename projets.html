Projet en cours !

Création d'un site de petites annonces

Le projet final qui nous a été commandé de faire dans la matière Applications Web et Culture Numérique consiste a créer un site de petite annonces, de notre choix (immo, emplois...)

Nous disposons de 2 mois pour le réaliser en entièreté et voici les langages utilisés :

  • HTML
  • CSS
  • PHP

Les consignes :

Pour réaliser un site de petites annonces, nous devons :

  • Créer une interface d'accueil du site, où seront répertoriées les offres
  • Les utilisateurs pourront consulter ces offres en cliquant dessus, ils accèderont à la page de description du job
  • Toutes les recherches seront entrés dans des formulaires, un pour rechercher les offres (titre du job lieu et titre de contrat)
  • Il sera également possible de poster des offres d'emploi, grâce à un formulaire
  • Ces formualaires seront traités en PHP, à l'aide d'un fichier .txt
